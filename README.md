# Job-Market-Analysis
Python-based Job Market Analysis using Pandas, NumPy, Matplotlib &amp; Seaborn with Data Cleaning, EDA and Data Visualization.

## Objectives
- Clean and preprocess job market data
- Perform Exploratory Data Analysis (EDA)
- Analyze salary distribution
- Identify top hiring industries
- Find the most in-demand skills
- Create meaningful visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Files
- Job_Market_Analysis.ipynb
- Dataset (.csv)
- Project Report (.pdf/.docx)
- Visualizations

## Key Insights
- Identified salary trends across different job roles.
- Analyzed the most demanded technical skills.
- Explored hiring patterns across industries.
- Visualized important trends using charts.

## How to Run
1. Clone the repository.
2. Install required libraries:
   pip install -r requirements.txt
3. Open the notebook in Jupyter Notebook.
4. Run all cells.
